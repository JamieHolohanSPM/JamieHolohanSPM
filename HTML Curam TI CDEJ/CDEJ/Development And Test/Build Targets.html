build -f non_source\msi_stage\build.xml msi -Dinstallshield.dir=C:\path\to\InstallShield\2013 -Dinstaller.version=1.0.5

Check out client\components\word-file-edit\lib\IBMCuramWordIntegrationAssistant.msi

build -f non_source\msi_stage\build.xml copyToSourceControl

Check in client\components\word-file-edit\lib\IBMCuramWordIntegrationAssistant.msi

Generate a new MSI installer for the Chrome Native Host application.

-Dinstallshield.dir should point to InstallShield 2013 install directory (use quotation marks where necessary)

-Dinstaller.version sets the version of the installed Host application. This should be incremented any time a new version is to be released.
