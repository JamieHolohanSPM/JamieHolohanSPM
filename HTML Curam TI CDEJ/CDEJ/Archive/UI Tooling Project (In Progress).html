Curam TI CDEJ
UI Tooling Project (In Progress)
Provide a toolkit that enables it's users to contribute quality UI content to an IBM Curam application in order to meet the challenges and needs of end users today and for the foreseeable future.
–
 

Where:

–Users = User Interface Developers & Product Line Managers
–Contribute = UI Developers would deliver Projects artefacts to release repository. Product Line Managers would manage, monitor, and report on UI content deliverables.
–UI content = UI content is any content that belongs to IBM that renders the user interface of a IBM curam application.
•
–IBM Curam application  = Any IBM application that is made by Curam. This could be a public facing application or otherwise.
–end users = Any user that uses a IBM Curam application. For example this could be a public citizen , system administrator, case worker, or some other type of user.
 or –foreseeable future = To be cognicent of new and emerging trends in User Interface development that would be of interest to ends users of IBM Curam applications.
–
