Coding Standards

TechInf have a coding standard that there should be no trailing whitespace in our Java source code. This is validated by !Checkstyle. Setup the following shortcut in Eclipse to quickly remove trailing whitespace:

  • Open the preferences dialog with the Window->Preferences menu.

  • In the preferences tree go to General->Keys.

  • In the list of commands select "Remove Trailing Whitespace". (Use the search filter to find this quickly)

  • Set the focus to the "Binding" input box.

  • Press the key sequence you wish to use for removing trailing whitespace. (e.g. Ctrl+Alt+R does clash with any of the in-build Eclipse shortcuts).

  • A When dropdown list will then appear, set this to "In Windows".
