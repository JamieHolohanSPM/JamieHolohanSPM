WebSphere Application Server installation
  1. Install the IBM Installation Manager (if you don't have it already)
    \\tiny.mul.ie.ibm.com\Applications\WAS\8.0\InstallationManager
     
  2. Open the Installation Manager, Navigate on File>Preferences...
     
  3. Add a new Repository entry specifying the repository.config file
    \\tiny.mul.ie.ibm.com\Applications\WAS\8.5.5\Windows\Installer\repository.config
     
  4. Copy in your machine the FixPack from
    \\tiny.mul.ie.ibm.com\Applications\WAS\8.5.5Fixpacks\FixPack 5\All Platforms
     
  5. Extract both and put the extracted  part2 inside the extracted part1
     
  6. In Installation Manager add a new Repository entry specifying the repository.config file in the extracted part1
    C:\8.5.5-WS-WAS-FP0000005-part1\repository.config
     
  7. Deselect all the other repositories except the two you have just added
     
  8. Click OK and navigate again in File>Preferences... This is what you should see
     
  9. Click on Install and you should see the WAS 8.5.5.5 as an available install package, select this and click Next
     
  10. Accept licenses
     
  11. Choose an Installation Directory as C:\Exclude\WAS8555
     
  12. Accept default English for “Translation”
     
  13. Accept default on "Install Packages - select features to install" panel and click next.
    Note: Don’t select “Sample application” here

     
  14. Review the Summary and click Install
     
  15. Select “None” under “Which program do you want to start” and click “Finish”
