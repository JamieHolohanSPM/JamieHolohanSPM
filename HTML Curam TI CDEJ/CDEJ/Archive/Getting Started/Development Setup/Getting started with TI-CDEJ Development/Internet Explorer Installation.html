Modern.IE

 

Some developers they found really handy to have available all the Internet Explorer versions through Virtual PC or Virtual Box pre-configured virtual machines.

Steps:

  1. Install Oracl Virtual Box
  2. Download and use one of the available pre-configured virtual machines from the official site.

 

Installation of Internet Explorer on Windows virtual machines

Simply download Internet Explorer target installers and upgrade or downgrade to obtain a particular target version.

 

Instructions to downgrade and uninstall an Internet Explorer version on Windows 7

Steps:

  1. Hit Start button
  2. Write in the search bar: "Uninstall a program" and hit enter.
  3. From the "Program and Features" panel click the "View installed updates".
  4. Right click on "Microsoft Internet Explorer X" or "Internet Explorer X".
  5. Hit Uninstall.

Is required later to restart the virtual machine.
