Prerequisites

  • Installation Manager v1.6.3.1 or higher installed.
    • See http://www-01.ibm.com/support/docview.wss?uid=swg27025142 for IM install packages.
  • Any previous versions of RSA/RSM uninstalled.
    • This can be done using the Uninstall option in Installation Manager.

Installation

  1. Start Installation Manager (Start > All Programs > IBM Installation Manager > IBM Installation Manager).
  2. Configure RSA repositories:
    1. Go to File > Preferences.
    2. Click Add Repository.
    3. Navigate to \\tiny.mul.ie.ibm.com\Applications\RSA\9.0\RSA\disk1\diskTag.inf and click OK.
    4. Click Apply and OK.
  3. Start the installation by clicking Install on the main screen of Installation Manager.
  4. Select the product you wish to install (IBM Rational Software Architect > Version 9.0) and click Next.
    • Warning regarding the Anti-Virus software can be safely ignored. Click Next.
  5. Accept the Licenses and click Next.
  6. Choose a location for the Shared Resources and click Next.
  7. Choose the package installation location, change the Architecture Selection to 32-bit and click Next.
    • The Architecture selection is important, as some of the plug-ins do not function correctly with the 64-bit version.
  8. Do NOT extend an existing version of Eclipse to avoid package conflicts. Click Next.
  9. On the "Install Packages" screen, make sure to select "Rational ClearCase SCM adapter" and "Rational Rose model import". Click Next.
  10. Continue and accept the defaults for the rest of the installation.

Post-Install Steps

  1. Choose "none" at the end of the installation, or else close the IBM Rational Software Architect window.
  2. Copy the 3 JAR files located in the directory below to RSAInstallDir\plugins\ 
    • JDE_ROOT\release\CuramSDEJ\rsa\eclipse\plugins\
      • where JDE_ROOT is the home of your stream, e.g. C:\CC\andreyzh_TI_MAIN\TIVOB
    • If the release\CuramSDEJ directory does not exist, run 'build server' from JDE_ROOT
  3. Create a new workspace directory in the RSAInstallDir corresponding to the stream you intend to work on, e.g. RSAInstallDir\ws_TI_MAIN
  4. Create/Edit the "IBM Rational Software Architect" windows shortcut on your desktop:
    • Change the shortcut name to include the stream you intend to work on, e.g. "IBM Rational Software Architect TI_MAIN".
    • Add the following two parameters to the Target field:
      • -clean
      • -data RSAInstallDir\ws_TI_MAIN
    • e.g.: C:\Exclude\IBM\RSA9.0\eclipse.exe -product com.ibm.rational.rsa.product.v90.ide -clean -data C:\Exclude\IBM\RSA9.0\ws_TI_MAIN  
    • Do this for every new stream that you intend to model

RSA License

Downloading the license

  1. Follow the instructions on the following page to create an account in IBM Rational License Key Center, if you don't already have one:
    • https://w3-connections.ibm.com/wikis/home?lang=en-gb#!/wiki/Wc8429a358499_452c_9958_623d8a360705/page/License%20Keys%20for%20Internal%20Use
  2. Log into the IBM Rational License Key Center: https://licensing.subscribenet.com/control/ibmr/login
  3. From the product line menu, navigate to IBM Rational Software Architect.
  4. Select the activation kit of the version of RSA you are using and download the license.

Installing the license

  1. Extract the contents of the zip into a new folder.
  2. Start IBM Installation Manager and select "Manage Licenses".
  3. Select the IBM Rational Product, then choose the "Import product activation kit" option.
  4. Browse to the jar file that you unzipped in step 2 and click Next, then Finish.
  5. The activation kit will be installed.
    • To verify the license installation, you can select Manage License in IBM Installation Manager or view it through Help -> License status in the "Rational Product".
  6. The permanent license is now installed and you can continue using the Rational product installed.
