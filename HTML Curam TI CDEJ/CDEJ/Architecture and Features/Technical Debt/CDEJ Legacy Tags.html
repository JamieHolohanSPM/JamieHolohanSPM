CDEJ Legacy Tags

Below is the list of the legacy JSP tags used for the HTML rendering on our pages, corresponding wrappers and renderers if any, the information on the current usage and the recommended action.

A number of them is still used, some might be removed as they represent outdated and no longer supported functionality. Some others are wrapped to allow for using them without the proper converting into renderers.

This means that while they are included into the renderer system, they are pseudo-renderers, and most probably not efficient.
Some of the tags could have been already replaced by the renderers (subject to checking in the configuration), some might have been re-implemented as renderers for dynamic UIM.

The actions as they could be seen:

1. Review the tags which are not longer being used, represent outdated/not supported functionality, remove them and the related utility classes.

2. Review the tags which are in place even though the functionality is already available in the form of renderer - these need to be checked in the configuration, removed as well.

3. Review the tags which are wrapped by the wrapper classes, these should take a priority when converting them to the renderers - if needed at all.

4. Review the rest of the tags to set the priorities of their conversion; consider if there are any to be left as is.
