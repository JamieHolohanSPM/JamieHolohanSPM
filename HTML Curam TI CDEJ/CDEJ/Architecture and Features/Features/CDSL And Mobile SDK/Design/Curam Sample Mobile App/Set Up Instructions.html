Set Up Instructions

 

Install Worklight Studio

  1. In eclipse, go to Help -> Install New Software -> Add…
  2. In the Add Repository Dialog, click Archive…
  3. Select iws_eclipse_6.1.0.1.zip from the following location on tiny: \\tiny\ProductDevelopment\TechInf\Client_Infrastructure\Mobile
  4. Give it a name, Worklight Studio 6.1.0.1 (This is the version currently supported by the Curam Sample App.) Then click OK.
  5. Select “IBM Worklight Studio Development Tools” from Install New Software window. Click Next and Next. Accept the license agreements and Click Finish.
  6. Installing is now kicked off. Wait until the installation is finished successfully and restart the eclipse. Now the installation of Worklight Studio is complete.

 

Set up Curam Sample App Project

  1. Go to File -> Import… -> Existing Projects into Workspace -> Browse… Select the CuramSampleMobileApp project from \TIVOB\test location. Click Finish to Complete
  2. Switch Eclipse to Design Perspective. Navigate to CuramSampleMobileApp project -> apps -> CuramSampleMobileApp folder
  3. Right click on the CuramSampleMobileApp folder -> New -> Worklight Environment -> Select iPhone -> Click Finish to add an iPhone environment.
  4. Right click on the CuramSampleMobileApp folder again -> Run As -> Build All Environments, to start building all environment.
  5. Once again, right click on the CuramSampleMobileApp folder -> Run As -> Run on Worklight Development Server to start the Curam Sample Mobile App on the Worklight.

 

Run Sample App in Mobile Simulator

  1. Right click on the CuramSampleMobileApp project -> Open Worklight Console. If the Worklight Console opens in the Eclipse internal browser, go to Windows -> Pereference -> General -> Web Browser -> Use external web browser.
  2. Click iPhone icon on the console to open the simulator
  3. Now you can see the Curam Sample Mobile App running in a iPhone simulator.
