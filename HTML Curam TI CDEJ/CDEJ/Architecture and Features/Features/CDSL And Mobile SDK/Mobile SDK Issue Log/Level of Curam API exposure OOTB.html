Level of Curam API exposure OOTB

Description: Should Curam expose mobile enabled APIs for use by customers?

Status: Closed

Assigned To: N/A

Resolution: We are not exposing any of our existing facades for use in mobile as they are not mobile streamlined (e.g. level of data returned, no pagination and no data-based security). Also, we know that we don’t have view on what our customers will need out of the 20k+ facades.

We will not enable our existing facades for mobile access outside of the sample.  Customers will develop their own specific mobile facades, using standard Cúram development and re-using functionality where available.
