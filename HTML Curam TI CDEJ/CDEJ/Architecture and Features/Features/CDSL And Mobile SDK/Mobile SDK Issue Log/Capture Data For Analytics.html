Capture Data For Analytics

Description: What data is relevant to capture for analytics purposes from CDSL ?

Status: Open

Assigned To: Susan and Niamh (threw question to research also)

 

Resolution: 

 

Regarding what kind of data would be relevant for analytics, I list some of them below and split them into two categories, targeted and batch. Targeted means that we would want to get a small number of specified entities (e.g. the basic info of a person or their household) and batch means that we would want to pull all the entities (e.g. we want all the providers in order to pick the best ones for a given plan). The list below is certainly not final.

Targeted:
-Persons (including address, contact info etc). Both clients and MDT members.
-Person relationships
-Assessments
-Plans
-Schedules

Batch:
-Providers
-(Perhaps) Payments
-(Perhaps) Schedules to optimize across cases.
