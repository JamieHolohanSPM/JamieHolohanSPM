Overview

The information below was gathered due to a customer request on what potential Google Map APIs they would need to subscribe to if they went into production with that feature enabled. The request was as a result of demo given to Norway in Jan 2020 where the integration of Google Maps was included.

Details

Note the information below may not be complete as the analysis performed was done in a very short space of time. However, it does give an idea on what Google Maps APIs are included and referenced across some of the SPM code streams.

OOTB Streams Referenced

  • CEF
  • CEFWidgets
  • SPM Ent Mods
  • HCR

Google Maps API Picker

  • https://developers.google.com/maps/documentation/api-picker

Google Maps APIs Used

1. Maps JavaScript API (https://developers.google.com/maps/documentation/javascript/tutorial) - Add a map to a web page. Customize the map with your own data and imagery.
Optionally support map controls, as well as gestures on the mobile web.

2. Directions service in the Maps JavaScript API (examples) (https://developers.google.com/maps/documentation/javascript/directions) - Get directions
data from origin to destination using various forms of transport: walking, driving, cycling, public transit.

Google Maps JavaScript APIs Referenced in OOTB Code

  1. https://maps.googleapis.com/maps/api/js?v=3&sensor=false&client="
  2. https://maps.googleapis.com/maps/api/js?v=3&sensor=false&key="
  3. https://maps.googleapis.com/maps/api/js?v=3&sensor=false&language="
  4. "http://maps.google.com/maps?f=d&saddr=" + fromAddress + "&daddr=" + destination + "&hl=" + language + "&pw=2";
