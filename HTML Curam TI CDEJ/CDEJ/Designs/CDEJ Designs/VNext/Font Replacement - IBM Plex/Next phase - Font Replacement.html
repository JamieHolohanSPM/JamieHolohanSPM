Next phase - Font Replacement

 

Minimum

TI - update font files and main-ibm-font.css
SPMEntMods - update CCV font files and main-ibm-font.css (noting that this css is not really css) and build the SWF
OneUI - prettify CSS (and check whether it gets minified in the build - if not, make prettification bare minimum number of additional chars by removing indentation etc), deliver, run automated scripts, delive

Optional

BizInf & HCR - introduce & document ieg-config.properties entries for the font-family to enable the removal of the literal font-families through IEG CSS

Bonus

Tech Debt - Replace overloaded font-faces with just 1 and put font-weight everywhere it is needed. This is better in many ways, one being we'd have no more faux-fonts and therefore a predictable look.
