  • Compares screenshots or HTML files with reference screen shots and reports visual differences on the page. 
  • Capturing Screenshots is slow. It took over 5 minutes to capture 320 pages.
    •  Screenshots not recommended for our count of 25000 pages.
  • BackstopJS uses puppeteer behind the scenes. Any additional Puppeteer scripts that are added will run before backstops comparison. 
  • Initial setup is not complex & if we generate HTML files we won't need to write complex scripts to access many pages.
    • HTML files won't be full renderings
    • Configure the JSON files for initial setup and add scenarios (as direct URL or scripted scenario).
  • Puppeteer scripts can be used on their own to generate a report showing where helvetica font types exist. It doesn't need backstopjs for that information.
  • Backstopjs tests  will only run on chromium.
