Multi-page view

We use multiple i-Frames on a single page to display different parts of the application and scan those pages at once using the font analyser script. HTML page previews can be created using the client-with-previews command. The idea will be to load as many previews as possible onto a single page so we can tell at a glance whether those pages still use helvetica font. Browsers vary on the number of iFrames they will support. Most should be able to handle 20 iframes. Each of the iFrames loaded on the page will need a specified height and width. Some font may be off the screen if the preview does not fit within the iframe.
