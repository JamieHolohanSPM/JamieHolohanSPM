Process for Migrating UIToolkit Tools

A new Ant script was created that will facilitate to download the dependencies needed, in this initial version just the Intern and Selenium are being updated.

 

The script uses NPM (that comes from NodeJS installation) to download the dependencies needed, zip them into a compressed file and send them into the same network location that CEF_MAIN and TI_MAIN streams use. So next time that you build your CEF_MAIN or TI_MAIN (if the same versions are configured in UIToolkit script, CEF and TI_MAIN) the correct dependencies will be used.

 

Below you can find an explanation on how to run the new script and where to configure the version for the dependencies.

 

First of all we need to have NodeJS installed in the machine and in order to download and install it please go to https://nodejs.org/ and follow the installation instructions.

 

After you have NodeJS installed in the machine you will be able to run the Ant script created to download the dependencies.

 

Configuring versions

 

  1. Navigate using explorer to your TI_MAIN\TIVOB\build_scripts folder
  2. Edit the file uitoolkit_properties.xml with the correct version of the dependencies that you need and save it

 

Running the Script

  1. Open a console window
  2. Navigate to your TI_MAIN\TIVOB folder
  3. Run the following command: build -f build_scripts\upgrade_uitoolkit.xml buildNpmDependencies
