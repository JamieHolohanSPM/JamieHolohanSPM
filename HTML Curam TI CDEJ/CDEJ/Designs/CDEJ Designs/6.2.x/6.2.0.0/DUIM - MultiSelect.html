DUIM - MultiSelect

Design Page for DUIM - MultiSelect

Background

Dynamic Evidence has to generate DUIM pages containing multiselect widgets in support of the functionality as specified in SPMP-14055 / RTC 122758. The MULTISELECT widget is not supported in Dynamic UIM prior to Curam 6.2. The enhanced new evidence screen will have a UI where we list all the case participants in a tabular form in the participant cluster. The case worker can select multiple participants to whom the evidence must be added and capture the evidence information. This will create an evidence record for each of the selected participants with the same evidence information.

Requirements

To implement support for the MULTISELECT widget on Dynamic UIM pages.

  • TEC-17787 / RTC 106996 - DUIM Support for multiselect widget

Documentation/How To Use Feature

  • Where the feature fits in the overall component?

    • Feature will enable support for the MULTISELECT widget in Dynamic UIM.

  • How it may be accessed (i.e. created/authored)?

    • The MULTISELECT widget in Dynamic UIM is constructed the same way as in static UIM pages.

  • What does it do (may be linked to the use cases above)?

    • It provide a multiselect option for a caseworker .

  • Advantages of using the feature?

    • Reducing the input for a caseworker. At the moment when for example new address needs to be added to participant it needs to be done for each participant living at the same address.

  • A description of how the feature should be used etc.? This may also include a description of a contract between the various teams using the feature and will incorporate some details of the design.
