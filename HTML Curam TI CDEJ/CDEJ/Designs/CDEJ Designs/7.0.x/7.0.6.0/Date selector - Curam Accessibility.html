Problem

Curam Accessibility - JAWS issue selecting the Month in the Calendar

Multiple issues found when Calendar was used to populate the Month field with the screen reader JAWS.

Use cases

 

These tests should be performed in all Scenarios from where you can Launch Select a Date Modal 

  1.  From a Page
  2.  From with in a Modal (ex: Register Person)

Requirements for accessibility use cases:

  1. Turn on the screenreaders voiceover or JAWS
  2. Use the keyboard to navigate in the screen
  3. Use the supported keys/key combinations or gestures if on tablet/desktop.
