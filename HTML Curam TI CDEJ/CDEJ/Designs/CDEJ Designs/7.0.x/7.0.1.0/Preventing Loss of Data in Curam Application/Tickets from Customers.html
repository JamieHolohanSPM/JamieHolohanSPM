Title: Searching for a participant or case results in "Do you want to leave this site?" pop-up

Steps to reproduce

Scenario #1

1.      Log in to the v7 Sandbox as Caseworker
2.      Select Cases and Outcomes
3.      Select ShortcutsàAll Participants Search  (any of the searches really.  Case search happens too)
4.      Select Search then Reset, and repeat until you get the error.  Occasionally just pressing Reset multiple times
NOTE: Sometimes it takes a while to happen, sometimes it happens immediately and often

Scenario #2

1) Using Chrome login as caseworker
2) Register a new person
3) Create a new Outcome Plan for that new person
4) Launch Firefox (or IE) and login as administrator
5) On FF, go to Admin Workspace-->Program Management->click on Services
6) Back on Chrome click on the Activities tab of the Outcome Plan that's opened
7) And create a new Service
8) While going through the wizard of creating a new Service (choose any from the Service list), while on a page, make sure you go back to FF and select Service to view.  
9)  Eventually, you will see the "Leave" pop-up.
